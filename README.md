# Portfolio Website for Mason

This is a **personal portfolio website** built for Mason to showcase their photography and videography work.

## About

I developed this website to provide a clean, modern, and responsive portfolio for Mason. It includes sections for featured work, client projects, visual stories, and image galleries.

## Features

- Responsive design for desktop and mobile  
- Image galleries and visual storytelling sections  
- Embedded reels and short films  
- Easy-to-navigate layout
